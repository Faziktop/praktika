# Руководство пользователя

## Две версии программы

### Версия 1: Управление (CRUD операции)
**Файл:** `main_crud.py`

**Функциональность:**
- Управление клиентами (добавление, просмотр, редактирование, удаление)
- Управление товарами (добавление, просмотр, редактирование, удаление)  
- Управление заказами (создание, просмотр, изменение статуса, удаление)
- Очистка базы данных
- Заполнение тестовыми данными

### Версия 2: Аналитика (Вычислительный эксперимент)
**Файл:** `main_analytics.py`

**Аналитические функции:**
- Общая статистика (количество заказов, клиентов, товаров)
- Популярные товары
- Средний чек и анализ чеков
- Лучшие клиенты
- Динамика заказов по месяцам
- Анализ выручки

## Запуск программ

### Версия управления:
```bash
cd database_version
python main_crud.py
